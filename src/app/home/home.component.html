PowerNSX

PowerNSX is a PowerShell module that abstracts the VMware NSX API to a set of easily used PowerShell functions.

Network automation - Power that is approachable

A familiar CLI

PowerNSX leverages PowerCLI to extend infrastructure automation across the entire stack. Be it scripting or application development - PowerNSX provides an extensive toolkit.

Building blocks

By covering the majority of the NSX for vSphere API, PowerNSX provides Create, Restore, Update, and Delete cmdlets for common Day One and Day Two Operations.

Infrastructure Blueprints

Your infrastrcture is now code. Utilise PowerNSX with PowerCLI to deploy entire network and application stacks encompassing routing, switching, security, and additional network services.

Angular

From development to production - your network is now code

Pipeline-based

Fundametal to Powershell is the pipeline. Easily retrieve objects and pass them along the pipeline for simplified interaction.

Open Source mandate

PowerNSX is Open Source. See the source code. Use the module. Contribute back. By being Open Source it helps to ensure rapid development and allows for contributions from the community.

Multi-platform

PowerNSX takes advantage of PowerShell Core. This allows PowerNSX to run on macOS, Linux, Windows, and within a Docker image. Run your code from wherever you develop it.

Reliable

Utilising Continous Integration and Deployment, the PowerNSX team is focused on shipping frequently, with a high bar of quality as standard. Automated tested via Pester help avoid regression.
