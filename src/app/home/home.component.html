PowerNSX

PowerNSX is a PowerShell module that abstracts the VMware NSX for vSphere API to a set of easily used PowerShell functions.

Network automation - Power that is approachable

A familiar CLI

PowerNSX is built to interoperate seamlessly with PowerCLI to extend infrastructure automation across the entire stack. Be it interactive operation, reporting, or complete scripting of entire stack deployments, PowerNSX provides a comprehensive and familiar toolkit.

Building Blocks

By leveraging the cmdlet based architecture of PowerShell, PowerNSX provides a modular approach to interacting with your NSX for vSphere deployment for all common NSX object creation, retrieval, configuration, and removal operations.

Forged by PowerShell

PowerShell serves as the foundation behind PowerNSX. PowerShell is an automation and configuration tool/framework that works well with your existing tools and is optimized for dealing with structured data and object models.

Automation

Use PowerNSX to impress your friends! Completely automate the standup and teardown of NSX for vSphere environments with PowerNSX. Code -> Run -> ... -> Profit!

From development to production - your network is now code

By the Power of the Pipeline

PowerNSX leverages the awesome power of the Powershell pipeline to easily create, retrieve, update and remove NSX configuration with easy to understand pipeline operations.

Open Source - Community Support

PowerNSX is Open Source. See the source code. Use the module. Contribute back. PowerNSX is supported by the community, for the community, encouraging rapid development. It is not an official VMware product.

Multi-platform

PowerNSX fully supports PowerShell Core as of version 3. This allows PowerNSX to not only run on macOS, Linux, Windows, or even from within a Docker image, but also provides the freedom to be able to develop scripts on any platform and run the same script unmodified on any other.

Reliable

Utilising Continuous Integration development enabled by a comprehensive Pester based testing framework, the PowerNSX team is focused on shipping frequently, and with a high bar of quality.
